**doraval** (*dor-uh-val*) = **Doraemon** + **eval**. A **context engineering toolkit** for coding agents.

Context you cannot rely on wastes a million tokens. Install the skill, then check with `dora` — yourself or via your agent.

## Start here

| Path | What you get |
| --- | --- |
| **[Audit my agent context](/get-started/audit/)** | Skill install + `dora review` steps only |
| **[Quickstart](/get-started/quickstart/)** | Full tutorial: skill, CLI, scan, review, fix, scaffold, memory |

```bash
# audit path (step 1)
npx skills add saif-shines/doraval

# then
npx @hacksmith/doraval review .
```

**doraval** (*dor-uh-val*) = **Doraemon** + **eval**. A **context engineering toolkit** for coding agents: install the skill first, then check context with the `dora` CLI (yourself or via your agent).

Context you cannot rely on wastes a million tokens. doraval makes verification **repeatable** — so you (and your agents) check before calling the work done, across Claude, Cursor, Codex, Copilot, and Grok.

## First win

**Step 1 — install the doraval skill** (required first step for agent-driven checks):

```bash
npx skills add saif-shines/doraval
```

**Step 2 — review context on this project:**

```bash
npx @hacksmith/doraval
# or after install: dora
```

Bare `dora` reviews agent context: surfaces, skill health, contradictions, next actions. Zero config. No API key. The skill uses the same engine when your agent runs checks.

```text
$ dora

  Health
    ✓ .claude/skills/review    valid
    ✗ .claude/skills/deploy    Missing "description"

  Next
    1. dora fix .claude/skills/deploy
    2. dora review --all
```

Common follow-ups: `dora review`, `dora fix`, `dora new --for`, `dora memory`, `dora reconcile`. Full table: [Command reference](/commands/scan/).

## What

Heuristic check that the body includes at least one fenced code block.

## Why

A five-line example beats a five-paragraph explanation. Skills that touch APIs, CLI flags, or file formats should show the shape of correct output, not only describe it.

## How to fix

Add a minimal, copy-pasteable example next to the rule it illustrates. Prefer one real snippet over many abstract paragraphs. Put large multi-file samples in `references/` or `assets/` if they bloat the body ([R007](/reference/rules/R007)).

## What

Parses the skill's frontmatter as YAML and reports malformed syntax.

## Why

Invalid YAML prevents agents from reading the skill's metadata. This rule is locked because malformed metadata makes later checks unreliable.

## How to fix

Correct the YAML syntax between the opening and closing `---` lines. Check indentation, quoting, and list formatting, then run `dora` again.

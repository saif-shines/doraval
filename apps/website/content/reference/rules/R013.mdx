## What

Checks that a skill's `scenarios.yaml` file can be parsed and has the expected scenario structure.

## Why

Scenarios define expected trigger behavior. Invalid scenario data cannot be used to evaluate whether the skill activates in the right situations.

## How to fix

Correct the YAML syntax and required fields in `scenarios.yaml`. Keep each scenario focused on one expected trigger or non-trigger case.

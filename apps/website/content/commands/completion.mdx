Tab completion is **install plumbing**, not part of the scan → fix loop. There is no `dora completion` command.

```bash
dora --completion bash
dora --completion zsh
dora --completion fish
```

Pipe or save the printed script into your shell’s completion directory (or eval it from your rc file).

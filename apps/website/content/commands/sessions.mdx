```bash
dora sessions
dora sessions --agent claude --limit 20
dora sessions show <id>
```

Read-only. Adapters ship for Claude Code, Grok, Cursor, Codex, and Copilot, so sessions from every one show up here.

## Options (list)

| Flag | Meaning |
| --- | --- |
| `--agent <name>` | Filter (`claude`, `grok`, `cursor`, `codex`, `copilot`) |
| `--limit <n>` | Max sessions per agent (default `10`) |
| `--format` / `--ci` | Machine output |

## show

Timeline: user turns, tool calls (names + args summary), skills invoked, token totals when available.

```bash
dora sessions show <sessionId>
```

```bash
dora providers
dora providers --json
```

Static packaging/spec matrix: manifest paths, marketplace layout, MCP filename per agent.

**Not** “which agents does this repo use?” — that is bare [`dora`](/commands/scan/) (Agent surfaces).

Scaffold with `dora new --for <agent>`.

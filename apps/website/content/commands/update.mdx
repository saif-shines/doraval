`dora update` checks for and installs a newer doraval. Use `--check` when you only want availability; use `--via` in scripts to force the install channel.

```bash
dora update
dora update --check
dora update --yes
dora update --via npm
```

| Flag | Meaning |
| --- | --- |
| `--check` | Only check; do not install |
| `--yes` | Skip confirmation |
| `--via homebrew\|npm\|bun` | Force install method (scripts / CI) |

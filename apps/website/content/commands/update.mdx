```bash
dora update
dora update --check
dora update --yes
dora update --via npm
```

| Flag | Meaning |
| --- | --- |
| `--check` | Only check; do not install |
| `--yes` | Skip confirmation |
| `--via homebrew\|npm\|bun` | Force install method (scripts / CI) |

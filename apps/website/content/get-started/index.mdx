**doraval** is a **context engineering toolkit** for coding agents. Make agent context work on every try: skills, plugins, rules, and decisions you can rely on across Claude, Cursor, Codex, Copilot, and Grok.

**First step for everyone who uses a coding agent:** install the doraval skill. Then check context with the `dora` CLI — yourself or through that agent.

## Why this exists

Agent context goes bad in ordinary ways:

- A skill ships with broken frontmatter or a vague description and never fires
- Claude and Cursor disagree on the same convention
- A decision lived in chat and is gone next session
- Always-on files grow until nothing is high-signal

You can fix each of those by hand. doraval makes the check **repeatable** — before someone (or an agent) calls the work done.

## Start here

```bash
# 1) first step — install the doraval skill
npx skills add saif-shines/doraval

# 2) review context on this project
npx @hacksmith/doraval
```

## What you can do with it

| Situation | Tool |
| --- | --- |
| What's broken in this project's agent setup? | `dora` / `dora scan` |
| Is this skill / plugin / rule good enough to ship? | `dora review` |
| Fix the mechanical issues (frontmatter, structure) | `dora fix` |
| Scaffold a skill, rule, agent, or plugin | `dora new --for <agent>` |
| Keep a principle and enforce it later | `dora memory` |
| Claude and Cursor contradict each other | `dora reconcile` |

Full flags: [Command reference](/commands/scan/).

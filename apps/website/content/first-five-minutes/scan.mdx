Your first CLI command is the default. In under a minute you learn which agents are configured, which skills are unhealthy, and the exact next command to run.

```bash
dora
# same as:
dora scan
```

Read-only. No writes, no LLM. Output covers:

1. **Agent surfaces**: which agents look configured (`CLAUDE.md`, skills dirs, …)
2. **Health**: skill and manifest issues with short reasons
3. **Next**: numbered commands to run next (`dora fix …`, `dora review …`)

Useful flags:

```bash
dora --format json
dora --ci                 # implies JSON
dora --cwd /path/to/repo
```

Full reference: [dora scan](/commands/scan/).

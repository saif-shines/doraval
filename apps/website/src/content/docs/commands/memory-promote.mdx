---
title: memory promote
description: Write high-weight principles into AGENTS.md (diff + confirm).
sidebar:
  order: 4
---

Compile hard rules into a managed `AGENTS.md` section so every agent shares them.

```bash
dora memory promote              # interactive confirm
dora memory promote --dry-run    # show diff only
dora memory promote --yes        # apply without prompt
dora memory promote --weight 8   # only weight ≥ 8 (default min is 7)
```

Always shows a diff before writing (unless `--yes` in non-interactive use with care).

See also: [memory add](/commands/memory-add/), [Memory](/concepts/memory/).

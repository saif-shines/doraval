`dora cursor new` is a context-aware wizard that scaffolds a new skill or plugin for Cursor, following its packaging conventions.

```bash
dora cursor new
dora cursor new my-skill
dora cursor new --yes --intent distribute my-plugin
```

The interactive flow and outcomes are identical to [`dora claude new`](/commands/claude-new/) — context detection, intent selection (self / self-later / distribute), and scaffold. The difference is the manifest location:

- **Plugin path**: creates `.cursor-plugin/plugin.json`
- **Standalone path**: creates skills under `.cursor/skills/`

## Options

| Flag | Description |
|------|-------------|
| `--yes` | Skip interactive prompts (use defaults and flags) |
| `--intent` | `self` \| `self-later` \| `distribute` |

## After scaffolding

```bash
dora validate . --for cursor
dora validate . --for cursor:plugin
```

See [`dora claude new`](/commands/claude-new/) for the full flow description and outcome examples.

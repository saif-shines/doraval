Turns `~/.doraval/memory/repo` into a real git clone (not the GitHub Contents API).

```bash
dora memory sync
dora memory sync --repo you/dora-memory
dora memory sync -m "capture principles from sprint"
```

**First run:** checks `gh auth`, creates a private `{you}/dora-memory` repo if needed, adopts local files, commit + pull --rebase + push.

**Later runs:** commit local changes, pull --rebase, push. Concurrent machines union-merge append-only principle files.

## Flags

| Flag | Description |
|------|-------------|
| `--repo <owner/name\|url>` | Remote (default `{gh-user}/dora-memory`) |
| `-m, --message` | Commit message when there are local changes |
| `--format` / `--ci` | Machine output |

See also: [Memory](/concepts/memory/).

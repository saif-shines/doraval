---
title: memory stash
description: Copy a gitignored or untracked file into project memory.
sidebar:
  order: 5
---

Keep local notes or scratch configs that git ignores but you do not want to lose.

```bash
dora memory stash notes.md
dora memory stash              # interactive picker (TTY)
```

Warns above 5MB per file; refuses above 50MB (use git-lfs). Interactive picker caps at 20 candidates and says when more exist.

See also: [memory restore](/commands/memory-restore/), [memory sync](/commands/memory-sync/).

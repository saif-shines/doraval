Output active memory so agents and docs stay aligned.

```bash
# Plain text (default)
dora memory context

# Full bodies
dora memory context --full

# Claude SessionStart hook JSON
dora memory context --json --quiet

# Maintain a managed section in a file
dora memory context --append-to CLAUDE.md
dora memory context --append-to AGENTS.md
```

## SessionStart hook

If you previously used a journal hook, point it at memory:

```bash
dora memory context --json --quiet
```

## Flags

| Flag | Description |
|------|-------------|
| `--full` | Full principle body |
| `--json` | Claude SessionStart `hookSpecificOutput.additionalContext` shape |
| `--quiet` | No plain-text noise when empty (hooks) |
| `--append-to <file>` | Upsert a managed section in that file |

See also: [memory promote](/commands/memory-promote/), [Memory](/concepts/memory/).

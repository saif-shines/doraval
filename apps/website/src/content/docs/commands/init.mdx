`dora init` is **gone**. Decision memory no longer needs a separate init step.

| What you wanted | Use instead |
|-----------------|-------------|
| Capture principles | [`dora memory add`](/commands/memory-add/) |
| List principles | [`dora memory list`](/commands/memory-list/) |
| Agent hooks / CLAUDE.md injection | [`dora memory context`](/commands/memory-context/) |
| Private backup | [`dora memory sync`](/commands/memory-sync/) |
| Eval / agent settings | [`dora config`](/commands/config/) |

Mental model: [Memory](/concepts/memory/).

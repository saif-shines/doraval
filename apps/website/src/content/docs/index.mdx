  <TabItem label="macOS (Homebrew)">
    ```bash
    brew install saif-shines/tap/doraval
    doraval validate .
    ```
  </TabItem>
  <TabItem label="npm / npx">
    ```bash
    npx @hacksmith/doraval validate .
    # if Bun is not installed: npm i -g bun
    ```
  </TabItem>
  <TabItem label="Bun">
    ```bash
    bunx @hacksmith/doraval validate .
    ```
  </TabItem>
</Tabs>

Keep current:

```bash
doraval update --check
```

List providers and keyword support for discovery:

```bash
doraval providers
```

Scaffold a skill or plugin:

```bash
doraval claude new --yes my-skill
doraval codex new --yes my-skill
doraval cursor new --yes my-skill
doraval copilot new --yes my-skill
```

Generate shell completions:

```bash
doraval completion bash
```
